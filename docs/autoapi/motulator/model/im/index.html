motulator.model.im

This module contains continuous-time models for induction motors.

Peak-valued complex space vectors are used. The space vector models are implemented in stator coordinates. The default values correspond to a 2.2-kW induction motor.

class motulator.model.im.InductionMotor[source]

Induction motor model.

An induction motor is modeled using the Gamma-equivalent model [1]. The model is implemented in stator coordinates.

Parameters

  • p (int) – Number of pole pairs.

  • R_s (float) – Stator resistance.

  • R_r (float) – Rotor resistance.

  • L_ell (float) – Leakage inductance.

  • L_s (float) – Stator inductance.

Notes

The Gamma model is chosen here since it can be extended with the magnetic saturation model in a staightforward manner. If the magnetic saturation is omitted, the Gamma model is mathematically identical to the inverse-Gamma and T models [1].

References

1(1,2)

Slemon, “Modelling of induction machines for electric drives,” IEEE Trans. Ind. Appl., 1989, https://doi.org/10.1109/28.44251.

class motulator.model.im.SaturableStatorInductance[source]

Stator inductance saturation model.

This saturation model is given by [2]:

L_s(psi_s) = L_su/(1 + (beta*abs(psi_s)**S)
Parameters

  • L_su (float) – Unsaturated stator inductance.

  • beta (float) – Positive coefficient.

  • S (float) – Positive coefficient.
