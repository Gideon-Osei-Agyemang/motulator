motulator.model.sm

Continuous-time models for synchronous motors.

The motor model can be parametrized to represent permanent-magnet synchronous motors and synchronous reluctance motors. Peak-valued complex space vectors are used. The default values correspond to a 2.2-kW permanent-magnet synchronous motor.
