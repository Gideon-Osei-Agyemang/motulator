class motulator.control.sm_obs_vhz.SensorlessFluxObserver(pars)[source]

Sensorless stator flux observer.

This observer is a variant of [1]. The observer gain decouples the electrical and mechanical dynamics and allows placing the poles of the corresponding linearized estimation error dynamics. For simplicity, the current model is here implemented in rotor coordinates, however this is mathematically equivalent to controller coordinates implementation in [2].

Parameters

pars (SynchronousMotorObsVHzCtrlPars) – Control parameters.
