motulator.control.sm_signal_inj

Sensorless control with signal injection for synchronous motor drives.

This module contains a simple example of square-wave signal injection for low- speed operation. A phase-locked loop is used to track the rotor position. For a wider speed range, signal injection could be combined to a model-based observer. The effects of magnetic saturation are not compensated for in this version.

class motulator.control.sm_signal_inj.SignalInjection(pars)[source]

Estimate the rotor position error based on signal injection.

This signal injection method estimates the rotor position error based on the injected switching frequency signal, according to [1]. The estimate can be used in a phase-locked loop or in a state observer to robustify low-speed sensorless operation.

Parameters

pars (SynchronousMotorSignalInjectionCtrlPars) – Control parameters.

References

1

Kim, Ha, Sul, “PWM switching frequency signal injection sensorless method in IPMSM,” IEEE Trans. Ind. Appl., 2012, https://doi.org/10.1109/TIA.2012.2210175
