class motulator.helpers.Sequence(times, values, periodic=False)[source]

Sequence generator.

This represents a sequence generator. The time array must be increasing. The output values are interpolated between the data points.

Parameters

  • times (ndarray) – Time values.

  • values (ndarray) – Output values.

  • periodic (bool, optional) – Enables periodicity. The default is False.
